pandas.Series.count#

Series.count()[source]#

Return number of non-NA/null observations in the Series.

Returns:
int

Number of non-null values in the Series.

See also

DataFrame.count

Count non-NA cells for each column or row.

Examples

>>> s = pd.Series([0.0, 1.0, np.nan])
>>> s.count()
2
